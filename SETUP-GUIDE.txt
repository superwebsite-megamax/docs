## How to Use
1. Extract this zip into your BEYOND-ENTERTAINMENT project folder (REPLACE all existing files)
2. Make sure your showdown-wallpaper.png is in the `public/` folder
3. Make sure your video/audio files are in `public/uploads/movies/` and `public/uploads/music/`
4. Run: `npm install`
5. Run: `npm run dev` to test locally
6. Run: `git add .`
7. Run: `git commit -m "Fixed all issues - complete rebuild"`
8. Run: `git push origin main`
9. Vercel will auto-deploy!

## Important Notes
- The TMDB API key is included in the code
- Your video files (Anne-Marie 2002, Kerosene ZOMBIES 4, Alice Merton, My Shot Hamilton, P!nk Just Like Fire) should already be in `public/uploads/movies/`
- The prebuild script in package.json will auto-generate manifest files from your uploads
- Wallpapers: Place `showdown-wallpaper.png` in the `public/` folder
